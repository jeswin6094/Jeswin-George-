# Jeswin George — Portfolio

> Personal portfolio website of **Jeswin George**, a creative Video Editor & Photographer from Kerala.

## 🌐 Live Site
**[jeswin-george.github.io/Jeswin-George-](https://jeswin-george.github.io/Jeswin-George-)** *(after GitHub Pages is enabled)*

## ✨ Features
- Dark cinematic design with purple gradient accents
- Animated loader & custom cursor
- Typed text animation
- Scroll-reveal animations
- Animated skill bars
- Contact form
- Fully responsive (mobile-friendly)

## 📋 Sections
- **Hero** — Introduction with availability badge
- **About** — Bio, education timeline & freelance experience
- **Skills** — 6 skill cards with animated progress bars + tool tags
- **Work** — Portfolio grid showcasing project types
- **Contact** — Contact cards + message form

## 🚀 Hosting on GitHub Pages
1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set Source to **Deploy from branch → main → / (root)**
4. Save — your site will be live in ~2 minutes!

## 📸 Adding Your Photo
Place your photo as `assets/jeswin.jpg` in the repo.
